#  Tetris Game
The simple source code written in Unity 2021.3 {C#} 


# How to Play
 - LeftArrow(D), 
 - RightArrow(A), 
 - DownArrow(S),

 - MouseRighButton: Rotates tiles to right
 - MouseLeftButton: Rotates tiles to left
